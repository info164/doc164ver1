PyCharm commandes et raccourcis
-------------------------------

- Cliquer deux fois sur une chaîne de caractères :

   - Pour chercher la chaîne dans un fichier CTRL-F
   - Pour remplacer la chaîne dans un fichier CTRL-R
   - Pour chercher la chaîne dans tous les fichiers CTRL-SHIFT-F
   - Pour remplacer la chaîne dans tous les fichiers CTRL-SHIFT-R
   - Raccourcis clavier sur la fenêtre à gauche du projet

-  Comparer deux projets CTRL-D


-  Raccourcis pour le texte

   -  Dupliquer la ligne où se trouve le curseur CTRL-D

En faire plus avec PyCharm `Optimisez
PyCharm <https://openclassrooms.com/fr/courses/6951236-mettez-en-place-votre-environnement-python/7014887-optimisez-pycharm>`__.
