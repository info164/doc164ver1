Documentation du module 164 (29.03.2024)
----------------------------------------

But
~~~
Le but de ce module 164 est de faire un CRUD (Create Read Update Delete) sur les tables d'une BD.

-  Ma démo consiste en une interface (HTML, FLASK, Python, etc.) et une
   BD MySql.

   -  Il s’agit d’un gestionnaire de films et leurs genres associés.
   -  Soit un système CRUD (Create Read Update Delete) presque complet
      sur les 3 tables de la BD films : Soit “t_film”; “t_genre” et la
      table intermédiaire “t_genre_film”


Votre travail
~~~~~~~~~~~~~

- Installer tous les programmes nécessaires (:ref:`installer_logiciel`)
- Faire fonctionner la démonstration "Films" (:ref:`demo_om`)
- Faire une copie de la démonstration (:ref:`copier_mon_projet`)
- Importer votre base de données (DUMP) dans votre copie (:ref:`importer_bd`)
- Connecter votre BD avec vos paramètres du fichier ".env" (:ref:`fichier_env`)
- Dans le fichier "gestion_genres_crud.py"(Ligne 37,46,50) sur "t_genre", il faut utiliser vos requêtes de votre table (:ref:`modif_req_table`)
- Modifier la page html "genres_afficher.html"(Lignes 37,40,43) pour qu'elle affiche les champs de votre BD (:ref:`modif_req_table_html`)
- Lire le chapitre :ref:`eval_projet`
